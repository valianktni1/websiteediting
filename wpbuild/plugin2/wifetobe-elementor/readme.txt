=== Wife To Be - Website (Elementor) ===
Requires Plugins: elementor
Stable tag: 1.0.0

Imports the complete Wife To Be website (all pages) as Elementor-editable pages that render
pixel-identical to the original, with every byte of SEO/schema/llms preserved.

== Installation ==
1. Plugins > Add New > Upload Plugin > this .zip > Install > Activate (Elementor must be active).
2. On activation the whole site imports and Home is set as the front page.
3. If links 404: Settings > Permalinks > Save Changes. Then purge host/LiteSpeed cache.

== Preserved exactly ==
Per-page title/description/keywords/robots/canonical/Open Graph/Twitter + JSON-LD schema,
robots.txt, sitemap.xml, llms.txt and llms-full.txt (served at site root). Domain kept as
wifetobe.org. Each section is preserved as its original markup for a 1:1 look and edited in
Elementor as a block. Auto-refreshes on version change.
