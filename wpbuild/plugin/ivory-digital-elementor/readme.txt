=== Ivory Digital - Website (Elementor) ===
Requires at least: 5.9
Requires PHP: 7.4
Requires Plugins: elementor
Stable tag: 1.2.0
License: GPLv2 or later

Imports the complete Ivory Digital website (all 22 pages) into WordPress as Elementor-editable
pages that render pixel-identical to the original site, with the original SEO intact.

== Installation ==
1. WordPress > Plugins > Add New > Upload Plugin > choose this .zip > Install > Activate.
2. Make sure the free "Elementor" plugin is installed and active.
3. On activation the whole site imports automatically and Home is set as the front page.
4. If you had an earlier version installed, go to the "Ivory Digital" admin menu and click
   "Re-import / Reset Pages" so the fixed pages replace the old ones.
5. If links 404, go to Settings > Permalinks and click Save Changes once.
